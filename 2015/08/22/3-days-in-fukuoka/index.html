福岡でだらだらする

Posted on 2015-08-22 / Sat
#生活 #旅行
-- スポンサードリンク --

8/19〜22の間福岡に一人旅行に行きました。行き帰りにフェリーを使った3泊4日の行程でした。

2015-08-19（水）

今日から夏休み。大阪は雨がちらつく曇天模様。13時過ぎに家を出て、一度会社によってからフェリーで福岡に向かった。 南海電鉄泉大津駅で下車して送迎バスで泉大津港に辿り着いた。受付を済ませて阪九フェリー「いずみ」に乗り込んだ。めっちゃ綺麗。

18:30 / 出港

船のエンジントラブルだとかで定刻より約2時間遅れての出港となった。

20:15 / 食事を終えた

  • 地鶏のたたき
  • 明太子
  • 豚汁（中）
  • ごはん
  • 〆て1,360円

お高いけどびっくりするぐらい美味しかったので満足。ご飯おかわりした。気が大きくなっているのもあるのかもしれない。
写真撮るの忘れた！

21:00 / 風呂に入った

大浴場に行ってきた。とても綺麗で広くて良い湯だった。

2015-08-20（木）

5:00 / 寝そびれた

うっかりPCで作業してたら朝だった。デッキに出てみたが曇りで日は見えなかった。

5:30 / 風呂に入ってきた

露天風呂を堪能した。

6:30 / フェリーで朝食

  • 塩さば定食
    • 500円

やっぱりご飯おかわりした。

7:00 / 上陸

船から降りて九州の地を踏んだ。
新門司港から小倉駅まで送迎バスで送ってもらい、そこからJRで博多に向かった。車内で熟睡して博多をひと駅通り過ぎてしまった。

8:30 / 博多到着

駅ナカをうろつき、ロッカーに荷物を預けた。なんとなく駅前のバス乗り場から適当なものに乗り込んだ。

天神あたりをうろうろ

バスで適当に降りたらそこは天神エリアだった。目当てはなにもなくひたすらうろついた。

12:30頃 / 福岡市立美術館

また適当にバスに乗った。しばらくして車内でGoogle Mapを見ると美術館に接近していたので慌てて降車ボタンを押した。 そこは大濠公園の中の福岡市立美術館だった。「浮世絵の世界」の展示を見た。めちゃよかった。照明が暗く、じっくり見すぎて目が疲れた。

14:00頃 / 大濠公園

展示をじっくり見た後は大濠公園の池の周りを散策した。気がすんだらやっぱり適当なバスに乗り込んだ。

15:00 / 西新駅

乗ったバスは博多駅とは逆方向に向かうものだった。
適当なところで降りてちょっとうろついて地下鉄空港線の西新駅から電車に乗って博多に向かった。

15:30 / 博多駅

乗り換えを間違えたりして手こずったがなんとか博多駅に戻ってきた。
駅ナカの土産屋で明太子を買って実家に送った。

  • 明太子2種 送料含み2,800円
    • 高え。おばちゃんの巧妙なトークに引っかかってしまった

歩く

博多駅から徒歩で海の近くのホテルに向かうことにした。
豪雨にたたられ濡れた。また携帯の充電が切れ、最後はほぼ勘で歩いた。靴ずれした…。疲れた。
途中の商店街で長浜ラーメンっぽいものを食べた。

  • 味噌とんこつラーメン
    • 430円

16:30 / ホテル到着

今夜の宿、ホテル博多プレイスに到着した。
コンドミニアム型つってほとんどワンルームマンションの1室と変わらないつくりだった。中も綺麗で気に入った。
ちょっとベッドに横になったら睡魔が襲って来てそのまま寝た。

20:30 / 夕食を求めて彷徨う

起きてちょっとしてから夕食を取るためにホテルを出た。

22:00 / やっと夕食にありつく

モツ鍋楽天地天神店

  • モツ鍋一人前
  • ちゃんぽん一人前
  • ハイボール
  • 1,500円

23:00 / シメ

  • カンパチ出汁茶漬け 980円

2015-08-21（金）

11:00 / チェックアウト

曇りだった

11:30 / バークレーでカレー

西新のあたりの商店街をぶらついていて目についた喫茶店？カレー屋さん？の「バークレー」に入った。

  • ハンバーグカレー 880円

結構感動的な美味しさだった！優しい辛さが新鮮だった。またいつか来たい。

12:00 / 博多駅

歩いて博多駅まで歩いたら暑くてしょうがなかった。 駅ナカで涼みがてら福岡の観光名所を探していると小倉に城があるらしいことが分かった。 ちょうどフェリー乗り場の行きしなに寄ることができそう。

13:15 / 西小倉

小倉城をぶらついた。城の中は近代化されていてあまり時代を感じることはなかった。 その後近所にあった図書館/文学館を冷やかした。

15:00 / 小倉

駅ナカの喫茶店に入って一息ついた。16時過ぎに出て駅前のバス停からフェリー乗り場までの送迎バスに乗り込んだ。 やたらと人が多く、バスは2台編成だった。私は座れず、小一時間立ったままだった。

17:10 / フェリーに乗り込む

帰りの船は「ひびき」だった。やっぱり綺麗で快適だった。

18:00 / 食事する

  • ステーキ丼
  • 揚げ餃子
  • 豚汁（小）
  • 〆て1,550円

また豪勢にいってしまった。けれどやっぱり美味しいので問題ないことにする。窓際の席に座って海を眺めながら食べた。 ちょうど食べ終わった頃に出港した。

21:00 / 入浴する

いいお湯だった。高校球児の一団と一緒になってしまい、彼らに比べて貧弱な自分の肉体を恥じた。

22:00 / 間食

なんだか小腹が減って再び食堂に吸い寄せられてしまった。

  • 炒飯 650円

うまかった。

24:00 / 就寝

この日はちゃんと寝た。電気を消して横になると船の揺れを感じられて心地よかった。

2015-08-22（土）

5:00 / デッキに出る

朝風呂した後デッキに出てみた。薄ぼんやりと明るくなってきている朝方の明るさだった。 この日も曇りだったので日の出は拝めなかったがうっすら空が赤くなっているのは見えた。 水平線に癒やされた。

5:30 / 朝食

  • 朝カレーセット 500円 なかなか美味しかった。阪九フェリーの食事には大満足

6:30 / 下船

下船して南海泉大津駅までの送迎バスに乗り込んだ。別の路線のバスに乗った人が多かったのか、こちらはがら空きだった。 駅についたら南海からJR環状線を乗り継いで帰宅した。

特別なことは何もしない、日常と地続きの一人旅が終わった。リフレッシュはできたと思う。
